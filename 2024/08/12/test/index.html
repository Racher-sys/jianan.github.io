什么是 Hexo

官网的解释：“A fast, simple & powerful blog framework, powered by Node.js”。

首先它是一个博客框架，其次它是使用 Node.js 编写的。

其特点是：

  1. 快速生成（我猜这是快速生成博客文件，是蛮快的，一行命令即可）
  2. 支持 GitHub Flavored Markdown 和大多数 Octopress （它也是一个博客框架）插件
  3. 一条命令即可部署到 GitHub Pages、Heroku 等。
  4. 强大的 API 可实现无限扩展
  5. 数百个主题和插件
